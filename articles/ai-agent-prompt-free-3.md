## プロンプト1: 競合調査（5分で市場把握）

**こんなときに使う**: 新規事業・新商品を検討するとき、市場の競合を素早く把握したいとき。

```
あなたは優秀なリサーチャーです。
以下の条件でサービスの競合調査を行い、表形式でまとめてください。

【調査対象サービス/市場】: {サービス名または市場}
【調査項目】: サービス名、価格帯、主な機能、ターゲット、強み、弱み
【調査件数】: 上位5〜8件

最後に「競合の共通課題」と「差別化できそうな点」を3つずつ挙げてください。
```

`{サービス名または市場}` の部分を自分の調べたい内容に変えるだけで動きます。

## プロンプト2: 会議メモ整理（貼るだけで議事録完成）

**こんなときに使う**: 打ち合わせ後、散らかったメモをすぐに整理したいとき。

```
以下の会議メモを整理してください。

【メモ】:
{会議メモをここに貼り付け}

出力形式:
## 決定事項
## 課題・懸念点
## 次のアクション（担当者・期日付き）
## 共有すべき情報

箇条書きで簡潔にまとめ、冗長な表現は省いてください。
```

## プロンプト3: ビジネスメール（依頼文を30秒で作成）

**こんなときに使う**: 取引先への依頼メールをスラスラ書きたいとき。

```
以下の条件でビジネスメールを作成してください。

【送信者】: {自分の役職・名前}
【宛先】: {相手の役職・関係性}
【目的】: {例: ○○の打ち合わせをお願いしたい}
【伝えたい要点】: {箇条書きで}
【トーン】: {例: 丁寧、フランク、フォーマル}

件名・本文・締め・署名のセットで出力してください。
```
