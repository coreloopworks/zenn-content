## 「AIに聞いたけど微妙な答えだった」は、ほぼプロンプトが原因

ChatGPTやClaudeを使っていて、こんな経験はないでしょうか？

- 「○○について教えて」→ 教科書みたいな当たり前の回答
- 「メールを書いて」→ テンプレ感が強くてそのまま使えない
- 「改善案を出して」→ 抽象的すぎて実行できない

実はこれ、AIが「賢くない」のではなく、**プロンプトの設計が足りていないだけ**です。

この記事では、AIの出力品質を劇的に上げる3つの改善法を具体例つきで紹介します。

---

## 改善法①: 「役割」を与える

**ビフォー:**
```
競合調査のやり方を教えて
```

**アフター:**
```
あなたは10年以上のキャリアを持つビジネスアナリストです。
新規事業を検討している私に対して、競合調査の進め方を
ステップごとに、具体的なツール名を含めて教えてください。
```

役割を指定するだけで、回答の「視点」と「粒度」が変わります。
「経験豊富な専門家として答えてください」と付けるだけでも効果があります。

---

## 改善法②: 「出力形式」を指定する

AIはデフォルトで「なんとなく適当な形式」で答えます。
自分が使いやすい形を明示するだけで、加工の手間がゼロになります。

**例:**
```
以下の条件でビジネスメールを作成してください。

【送信者】: 営業担当・田中
【宛先】: 新規取引先の部長
【目的】: 来週の打ち合わせ日程を調整したい
【伝えたい要点】:
  - 今月中に1時間お時間いただきたい
  - オンライン・対面どちらでも可能

出力形式: 件名・本文・締め・署名のセットで出力してください。
```

「表形式で」「箇条書きで」「3行以内で」など、**使う場所を想定した形式**を指定するのがコツです。

---

## 改善法③: 「制約条件」を加える

AIは指示が曖昧だと「無難な最大公約数」を出してきます。
制約を絞ることで、自分の状況にフィットした答えが出てきます。

**曖昧な例:**
```
SNS投稿を書いて
```

**制約あり:**
```
以下の条件でTwitter(X)の投稿文を書いてください。

【商品/サービス】: AIプロンプトテンプレートパック
【ターゲット】: 毎日ChatGPTを使うビジネスパーソン
【目的】: 無料記事への誘導（リンクは後で追加）
【制約】:
  - 140文字以内
  - 「悩み→解決策→CTA」の構成
  - 絵文字を2〜3個使う
  - 専門用語は使わない

3パターン出力してください。
```

「○○しないで」「○文字以内で」「○つだけ出力して」という制約ほど、出力が絞られて使いやすくなります。

---

## 3つの改善法をまとめると

| 改善法 | 効果 | 具体的な言い回し |
|--------|------|----------------|
| 役割を与える | 視点・専門性が上がる | 「あなたは〜の専門家です」 |
| 出力形式を指定 | 加工の手間がゼロになる | 「○○の形式で出力して」 |
| 制約条件を加える | 状況にフィットする | 「○文字以内で」「○パターンで」 |

この3つを組み合わせるだけで、同じAIツールでも**出力品質が大きく変わります**。
