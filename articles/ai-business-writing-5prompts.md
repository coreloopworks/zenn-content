## 「書く仕事」に毎日2〜3時間かかっていませんか？

議事録の清書、丁寧なビジネスメール、週次報告書、企画書のたたき台——。

これらは「重要ではあるが、ちゃんとやると時間がかかる」タスクの代表格です。ChatGPTやClaudeを使えば、**下書きを5分以内で生成し、確認・修正だけに集中**できるようになります。

この記事では、ビジネス文書を爆速で仕上げる実務プロンプトを5本、無料で公開します。

---

## プロンプト1: 議事録の清書（箇条書き→完成文書）

**使い場面**: 会議メモをとったが、読める形に整えるのが面倒なとき。

```
あなたは優秀なビジネスアシスタントです。
以下の箇条書きメモを、正式な議事録形式に整えてください。

【会議名】: {会議名}
【日時】: {日時}
【参加者】: {参加者リスト}
【メモ】:
{箇条書きメモをそのまま貼り付け}

# 出力形式
- 会議の目的（1〜2行）
- 決定事項（番号付きリスト）
- 次回アクション（担当者・期限つき）
- 継続検討事項（あれば）
```

**コツ**: メモは汚くて大丈夫。「→」「?」「要確認」なども含めて貼り付けるだけでAIが整理してくれます。

## プロンプト2: ビジネスメール作成（状況説明→丁寧な文面）

**使い場面**: 言いにくいことを丁寧に伝えたいとき、お詫び・お断りメールを書くとき。

```
あなたは日本語ビジネス文章の専門家です。
以下の状況に合った丁寧なビジネスメールを書いてください。

【送信者の立場】: {あなたの役職・立場}
【相手】: {宛先の会社・役職}
【用件】: {メールの目的を1〜2行で}
【伝えたい内容】: {ざっくり箇条書きで}
【トーン】: {丁寧/フォーマル/やや柔らかい など}

件名から本文まで書いてください。
```

**コツ**: 「トーン」を指定するだけで、堅すぎ・柔らかすぎを調整できます。

---

## プロンプト3: 企画書のたたき台（アイデア→構成+骨子）

**使い場面**: 新規施策・プロジェクトの企画書を提案しなければならないとき。

```
あなたは経験豊富な企画担当者です。
以下のアイデアをもとに、社内承認を通すための企画書の構成と骨子を作成してください。

【テーマ】: {施策・プロジェクトの概要}
【背景・課題】: {なぜこれが必要か}
【ターゲット】: {誰のための施策か}
【期待効果】: {何が改善されるか}
【予算感】: {おおよその規模}

# 出力形式
1. 企画書タイトル（案3つ）
2. エグゼクティブサマリー（3行）
3. 現状分析と課題
4. 提案内容と実施ステップ
5. KPIと評価指標
6. リスクと対策
7. まとめ（承認を促す一文）
```

**コツ**: 「期待効果」を具体的な数字で書くほど、説得力のある企画書になります。

## プロンプト4: 週次・月次報告書（実績→読みやすい報告）

**使い場面**: KPIや作業実績をまとめて上司・チームに報告するとき。

```
あなたは分かりやすいビジネス報告書を書くプロです。
以下の実績データをもとに、読みやすい報告書を作成してください。

【報告期間】: {期間}
【部門・担当】: {あなたの所属・役割}
【実績データ】:
{数値・完了タスク・進捗などを箇条書きで}

【課題・問題点】:
{感じていること・うまくいかなかったこと}

# 出力形式
- 今期ハイライト（3点）
- KPI進捗（表形式）
- 課題と来期への対応方針
- 来期の重点テーマ（1〜3つ）
```

## プロンプト5: クレーム・苦情対応メール

**使い場面**: 顧客からのクレーム・お叱りメールに、誠実に返信したいとき。

```
あなたは顧客対応のプロです。
以下の状況で、誠実かつ関係を維持できるクレーム対応メールを書いてください。

【クレームの内容要約】: {何について怒っているか}
【こちらの落ち度の有無】: {あり／なし／一部あり}
【対応できること・できないこと】:
  - できること: {具体的対応}
  - できないこと: {理由とともに}

丁寧さを保ちながら、問題の所在を明確にし、関係継続を促す文体で書いてください。
件名・冒頭のお詫び・本文・締めを含む完成版を作成してください。
```

## まとめ

| プロンプト | 用途 | 節約できる時間 |
|---|---|---|
| 議事録清書 | 箇条書き→正式文書 | 20〜30分 |
| ビジネスメール | 状況説明→完成文 | 10〜20分 |
| 企画書たたき台 | アイデア→構成+骨子 | 1〜2時間 |

これらのプロンプトを日々の業務に組み込んで、「書く時間」を「考える時間」に変えていきましょう。
